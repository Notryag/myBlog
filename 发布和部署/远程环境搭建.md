# 初始化环境

## 远程服务器

需要通过vscode中的`ssh remote`插件
保存本地电脑的`ssh`到远程服务器-> 为了不每次登录服务器输入密码
本地生成sshkey -> 同下 git中生成sshkey
保存到服务器中的`.ssh/authorized_keys`中

## 安装git

```bash
# 安装git

sudo yum install git
# 生成ssh 在 ~/.ssh/id_rsa.pub
ssh-keygen -t rsa -C “10000@qq.com”
# 生成的内容填入github中
cat ~/.ssh/id_rsa.pub
```


完整命令

```bash
sudo yum update -y
sudo yum install git

```
